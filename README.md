# bigfoot-soap-collection

## Requirements

Node version 16.0 or greater, npm version 8 or greater, Vue3, Vite

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
